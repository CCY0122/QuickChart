# 快制图表

### 如此轻松快速的制作你的图表

  快制图表是一款帮您快速且方便的制作出图表的app，支持多种常见的图表如饼图、雷达图、折线图、柱状图等等，使用简单，支持导出为图片并分享，高度的定制性让您轻松制作出自己喜欢的样式。
